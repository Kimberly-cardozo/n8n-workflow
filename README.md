### A modular personal AI assistant built using n8n workflows with a Streamlit-based UI.

This assistant automates everyday productivity tasks including answering questions, managing your calendar, handling emails, taking notes, and tracking expenses — all from a simple web interface.

### Frontend:

Built with Streamlit for a clean and interactive user experience

### Backend Automation:

Powered by n8n workflows

### Integrates with:

LLM APIs (for reasoning and responses)

Calendar APIs (e.g., Google Calendar)

Email providers (e.g., Gmail)

Storage services for notes and expense tracking

```
User (Streamlit UI) 
        ↓ 
     Webhook 
        ↓ 
       n8n 
        ↓ 
 External APIs (LLM, Email, Calendar, Storage)
 ```
